1) index.jsp - приветствия, поля логина и пароля, кнопка "Смены пароля", кнопка "Войти"(переход на страницу поиска)
2) login.jsp - изменение логина, кнопка "Назад", кнопка "Войти"(переход на страницу поиска)
3) show.jsp - список сущностей, кнопка "Назад", кнопка "Выбрать содержимое" (не для студентов), для групп кнопка "Поставить оценки" (не для студентов), кнопка "Редактировать"(для админа), кнопка "Добавить"(для админа), кнопка "Удалить"(для админа), кнопка "Добавить Вуз"(для админа), кнопка "Посмотреть свои предметы"(для админа)  
4) subjects.jsp - список дисциплин, кнопка "Назад к поиску" (не для студентов), кнопка "Выбрать" (для студентов), кнопка "Выбрать группу" (не для студентов), кнопка "Редактировать"(для админа), кнопка "Добавить"(для админа), кнопка "Удалить"(для админа) 
5) progress.jsp -  прогресс студента (предмет - оценка, экзамен - оценка), а препод тут может выставить оценки, кнопка "Назад к группам", "Сохранить оценки" (не для студентов)
6) group_journal.jsp - показывает весь список студентов и их окончательные оценки по предмету, кнопка "Назад к поиску"
7) error.jsp - страница "ошибок"
8) university.jsp - есть возможность для админа добавить новый университет, кнопка "Назад", кнопка "Добавить"(для админа)
